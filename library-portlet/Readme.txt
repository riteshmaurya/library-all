Reemplazar el fichero Innofis.ovpn por el nuevo que adjuntamos.

#### WINDOWS ####

Se encuentra en -> C:\Program Files\OpenVPN\config

#### Linux ####

Se ubica en -> /etc/openvpn/

Ejecutar terminal en Linux y copiar el fichero en /etc/openvpn/ con:

sudo cp /midirectorio/Innofis* /etc/openvpn/

#### macOS ####

Remplazar el fichero en la ubicación donde se ejecuta el script.
